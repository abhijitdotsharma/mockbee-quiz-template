## Key Features
* Home with Questions based on Categories
* Rules Page
* MCQ Based Question
* Responsive
* Result Page with Score of a User

## Tech used
* React
* TypeScript
* Vanilla CSS + [Nou-CL](https://nou-cl.netlify.app/) Component Library
* React Context API
* React Router v6
